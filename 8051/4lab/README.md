# Изучение режимов энергосбережения

В рамках данной работы был написан код на языке C и Assembler, который вводит микроконтроллер в режим ожидания, выходит из него по прерыванию и спустя 16 прерываний меняет состояние диода

Прерывание генерируется переполнением таймера 0, настроенным в 16 битном режиме – это значит, что таймер совершает 65535 отсчётов перед обнулением. На первом рисунке можно видеть включение режима ожидания(idle). Видно, так же, что таймер 0 начал отсчёт:

![Первый рисунок](<1.png>)

На втором рисунке можно видеть выключение режима ожидания(idle), это произошло по прерыванию таймера:

![Второй рисунок](<2.png>)

В cnt, счётчике прерываний, было выставлено число 16, это значит, что перед тем как инвертировать состояние порта 1.6 должно пройти 16 прерываний. На деле, это порядка 4.1 с, судя по разнице регистров sec на двух последних рисунках. На этих же рисунках, можно видеть переключения состояния диодов:

![Третий рисунок](<3.png>)

![Четвёртый рисунок](<4.png>)
